# Retail Sales & Profit Analysis - SQL Project

## Project Overview

This project analyzes retail sales data using SQL to identify insights related to sales performance, profitability, product categories, and regional trends.

## Tools Used

- SQL Server
- SQL
- CSV Dataset

## Analysis Performed

- Sales Analysis
- Profit Analysis
- Region Analysis
- Product Analysis
- City Analysis

## Key Insights

- High sales does not always mean high profit
- Some product categories are less profitable
- Discount impacts profitability
- Regional performance differs significantly

## Objective

To improve SQL and data analysis skills using a retail dataset.
